# Instinct Robotics

Innovative computer vision solutions for industrial robotics, leveraging state-of-the-art simulation and robust technology.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key
3. Run the app:
   `npm run dev`
